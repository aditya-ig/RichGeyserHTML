풍부한 간헐천 비디오 슬롯에 오신 걸 환영합니다. 석유 리깅 산업 테마 게임입니다. 이 게임은 구싱 와일드 재스핀과 여러 흥미로운 기능으로 제공됩니다.

게임 소개

풍부한 간헐천은 10개의 고정된 페이 라인으로 플레이됩니다. 이 게임에는 3개 이상이 페이 라인의 맨 왼쪽 위치에서 시작하는 주변 릴에서 연속되는 경우 지급되는 8개의 일반 심볼이 있습니다.

구싱 와일드가 보너스를 재스핀합니다

화면에 나타난 2개 이상의 와일드 심볼이 ‘구싱 와일드’ 재스핀 기능을 트리거합니다. 트리거 와일드가 있는 릴 아래의 각 ‘석유 간헐천’이 와일드 위치까지 석유를 분출시켜서 트리거 와일드 심볼 아래의 모든 위치를 와일드로 바꿉니다. 상금이 보상되고 와일드가 1칸 아래로 이동한 상태로 릴이 재스핀됩니다. 재스핀 중에 획득한 후속 와일드는 재스핀을 리트리거합니다. 재스핀은 모든 와일드가 화면에서 사라질 때까지 계속됩니다.

플레이 방법

  • 판돈/코인 액면가를 선택합니다.

  • 게임 주기를 시작하려면 스핀 버튼을 클릭(데스크톱) 또는 터치(모바일)하세요.

베팅 설정

여기에서 플레이어가 게임의 판돈 값을 결정하거나, 다른 옵션을 선택할 수 있습니다
자동 진행 다양한 표준 및 연장된 자동 진행 옵션을 선택할 수 있고 시작할 수 있는 페이지를 실행합니다. 손해 한도도 포함될 수 있습니다.
게임 설정 여러 일반 설정 옵션을 표시합니다(예: 세션 중에 재생되는 사운드를 결정할 수 있는 플레이어 선택 가능 오디오 설정).
홈 현재 게임 세션에서 나가고(최종 프롬프트 이후) 운영자의 메인 게임 메뉴 페이지로 돌아갑니다.

기타 게임 정보

  • 다른 라인의 일치하는 승리는 함께 추가되며 플레이어에게 지급됩니다.

  • 베팅 - 현재 설정의 베팅 값을 표시합니다. 베팅 크기는 고정된 페이 라인의 개수로 곱해진 1 코인입니다.

  • 총 베팅은 코인 값에 라인 당 베팅을 곱한 값입니다.

  • 최대 베팅(데스크톱 전용) - 최대로 가능한 코인 값을 설정합니다.

  • 스핀 버튼 - 게임을 시작합니다.

  • 자동 진행 - 매번 스핀 버튼을 누를 필요 없이 게임을 플레이하도록 설정할 수 있습니다. 자동 진행 버튼을 눌러서 자동 진행할 라운드의 횟수를 선택합니다(또는 스핀 버튼을 길게 눌러서 자동 진행 메뉴 열기(모바일)). 게임 설정에서 자동 진행을 정지할 때 추가 조건을 선택할 수 있습니다. 일부 게임 클라이언트 버전의 손해 한도는 자동 진행 세션 중에 설정된 한도 이상 잃는 것을 막아줍니다. 자동 진행을 수동으로 멈추려면 자동 진행 정지 버튼을 누르세요.

  • 승리 - 현재 또는 이전 승리 지급의 승리를 표시합니다.
